# Rebel Funding — Trader Data Repository

> ⚠️ **Repo in making** — building tools to analyze prop-firm trading competition data and discover what separates winning traders from the crowd.

---

## What's Here

### Trader Trade Histories (`trader_trades/`)
- **3,441 Excel files** — every trade placed by every trader across 32 Rebel Funding competitions (Aug 2023 – May 2026)
- Each file = one trader's full trade log: symbol, direction, entry/exit, P/L, stop loss, take profit, duration
- Naming: `Competition-MM-YY_R{rank}_{trader}.xlsx`

### Price/Candle Data (`candle_data/`)
- **OHLCV candles** for 51 instruments (FX pairs, indices, commodities, crypto) from MetaTrader 5
- 4 timeframes: **M15, H1, H4, D1** — stored as `.parquet` files
- `symbol_map.csv` — maps broker symbol names to canonical names

### News Calendar
- Economic events scraped from Investing.com (High/Medium impact) for 11 countries
- Includes actual vs forecast values with beat/miss/inline labels
- Generated by `scrape_news_calendar.py`

---

## Scripts

| Script | What it does |
|---|---|
| `rebels_funding_scraper.py` | Downloads per-trader Excel files from rebelsfunding.com leaderboard (needs authenticated Chrome) |
| `scrape_news_calendar.py` | Scrapes economic calendar events from Investing.com |
